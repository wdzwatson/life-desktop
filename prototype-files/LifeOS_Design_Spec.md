# LifeOS · 设计规格书 v1.0
> 面向个人 / 小团队的本地离线桌面效率应用
> 技术平台：Electron · 适配 macOS / Ubuntu / Windows
> 界面语言：中文 / English（可切换）

---

## 一、产品定位与设计哲学

### 1.1 产品一句话定位
**LifeOS** 是一款运行在本地的全能个人工作台，将任务规划、视频收藏、笔记、电子书、小工具统一整合在一个界面内，数据完全本地存储，后期可扩展云同步。

### 1.2 设计原则
- **模块化**：每个功能区域独立，互不干扰，但共享统一的搜索、标签、归档体系
- **主题化**：支持四套视觉主题，用户可在设置中随时切换
- **可扩展**：小工具模块采用插件化设计，后期可按需增减
- **离线优先**：所有核心功能无需网络，仅视频下载/网站收藏需要联网

## 二、视觉主题系统

用户可在 **设置 → 外观** 中切换以下四种主题，切换实时生效：

| 主题名称 | 风格参考 | 主色调 | 特征描述 |
|----------|----------|--------|----------|
| **Minimal**（极简） | Notion / Linear | 白底 + 深灰文字 | 大量留白，无装饰边框，内容优先 |
| **Dense**（信息密度） | Obsidian / VS Code | 深色背景 + 彩色高亮 | 紧凑间距，代码感，适合重度使用 |
| **Card**（卡片化） | Trello / Apple Notes | 米白 + 柔和阴影 | 圆角卡片，色块标签，视觉层次丰富 |
| **Dark Tech**（暗黑科技） | 赛博朋克 / 终端风 | 纯黑 + 霓虹蓝/绿 | 发光边框，等宽字体，高对比度 |

每套主题均包含：主背景色、侧边栏色、强调色、文字色、边框色、卡片背景色六个设计 token。

## 三、整体布局结构

```
┌─────────────────────────────────────────────────────────┐
│  标题栏（自定义无边框，含窗口控制按钮）                        │
├──────┬──────────────────────────────────────────────────┤
│      │  顶部操作栏（搜索框 · 快捷操作 · 用户设置）              │
│  侧  ├──────────────────────────────────────────────────┤
│  边  │                                                  │
│  栏  │           主内容区域（随模块切换）                    │
│      │                                                  │
│  60px│                                                  │
│  宽  ├──────────────────────────────────────────────────┤
│      │  底部状态栏（当前模块 · 数据统计 · 通知提醒）             │
└──────┴──────────────────────────────────────────────────┘
```

### 3.1 左侧边栏（固定，宽 60px，悬停展开至 200px）

侧边栏分三个区域：

**上方：主模块导航**
```
🏠  仪表盘（Dashboard）
✅  任务（Tasks）
🎬  视频（Videos）
📝  笔记（Notes）
📚  书库（Books）
🔧  工具箱（Toolbox）
```

**中间：快捷入口（用户自定义 Pin）**
- 支持将常用笔记本、书单、工具 Pin 到侧边栏

**底部：系统操作**
```
🔍  全局搜索
⚙️  设置
❓  帮助
```

### 3.2 顶部操作栏
- 全局搜索框（支持跨模块搜索，快捷键 `Cmd/Ctrl + K`）
- 当前模块的快捷操作按钮（如"新建任务"、"新建笔记"）
- 右侧：通知铃铛 · 主题切换快捷按钮 · 语言切换（中/EN）

## 四、模块详细规格

---

### 模块 A：仪表盘（Dashboard）

**用途**：应用启动后的默认页面，提供全局概览。

**布局**：响应式网格卡片

**包含组件**：
- 📅 **今日日程卡片**：显示今天到期的任务列表
- ⏰ **最近提醒**：未来 3 天内的提醒事项
- 📖 **当前在读**：最近打开的电子书（封面 + 进度条）
- 🗒️ **最近笔记**：最近编辑的 5 条笔记摘要
- 🎬 **最近视频**：最近添加的视频收藏缩略图
- 📊 **数据统计**：本周完成任务数、笔记字数、阅读时长
- 🔧 **固定小工具**：用户选择的 1-2 个常驻小工具（如番茄钟）

**交互**：每张卡片可点击跳转对应模块；卡片支持拖拽重排。

---

### 模块 B：任务（Tasks）

**用途**：日常任务规划、提醒管理。

#### B.1 视图模式（顶部 Tab 切换）
| 视图 | 描述 |
|------|------|
| **今日** | 只显示今天的任务，配合时间轴 |
| **清单** | 全部任务的平铺列表，支持分组 |
| **看板** | Kanban 列：待办 / 进行中 / 已完成 |
| **日历** | 月视图，任务以事件形式显示 |

#### B.2 任务数据结构
```
任务
├── 标题（必填）
├── 描述（富文本，可选）
├── 截止日期 & 时间
├── 重复规则（每日 / 每周 / 自定义）
├── 优先级（🔴高 / 🟡中 / 🟢低）
├── 标签（多选，颜色标记）
├── 子任务列表（无限嵌套）
├── 关联笔记（链接到笔记模块）
└── 完成状态
```

#### B.3 提醒机制
- 系统原生通知（Electron `Notification API`）
- 提前提醒：5分钟 / 15分钟 / 1小时 / 自定义
- 托盘图标上显示今日未完成任务数

#### B.4 交互亮点
- 快速添加：顶部一行输入框，支持自然语言解析（"明天下午3点开会" → 自动填写时间）
- 拖拽排序任务优先级
- 批量操作（多选 → 批量完成/删除/打标签）

---

### 模块 C：视频（Videos）

**用途**：在线视频收藏下载 + 本地视频管理播放。

#### C.1 子模块 Tab
```
[收藏夹]  [下载队列]  [本地库]  [播放历史]
```

#### C.2 收藏夹
- 添加方式：粘贴 URL（支持 YouTube、Bilibili、Twitter/X、Vimeo 等主流平台）
- 展示：视频封面缩略图 + 标题 + 时长 + 来源平台图标
- 操作：收藏 / 下载 / 分组 / 打标签 / 跳转原地址
- 视图：网格卡片 / 列表两种切换

#### C.3 下载功能
- 底层：集成 `yt-dlp`（内置，无需用户配置）
- 下载设置：视频质量选择（最高/1080p/720p/仅音频）
- 下载队列面板：显示进度条、速度、剩余时间
- 下载完成后自动归入本地库

#### C.4 本地库
- 扫描并管理指定本地文件夹中的视频文件
- 支持格式：mp4 / mkv / avi / mov / webm
- 缩略图自动提取（截取第5秒帧）
- 视图：网格 / 列表；排序：时间 / 文件名 / 时长 / 大小

#### C.5 内置播放器
- 基于 `mpv` 或 `vlc` 内核
- 功能：进度拖动 / 音量 / 倍速（0.5x-3x） / 全屏 / 字幕加载
- 支持画中画（PiP）模式
- 记忆播放位置（下次打开继续）

---

### 模块 D：笔记（Notes）

**用途**：多格式笔记记录、编辑、分类、归档。

#### D.1 布局
```
[左：笔记本树] [中：笔记列表] [右：编辑区]
  200px           240px           剩余宽度
```
三栏均可折叠，支持专注模式（隐藏左右，全屏编辑）。

#### D.2 笔记格式支持
| 格式 | 编辑器 | 说明 |
|------|--------|------|
| **Markdown** | CodeMirror / Monaco | 实时预览，支持 GFM |
| **富文本** | Tiptap / Quill | 所见即所得，类 Notion |
| **纯文本** | 轻量文本框 | 无格式，极速记录 |
| **代码片段** | Monaco Editor | 语法高亮，多语言 |
| **手绘草图** | Excalidraw 内嵌 | 矢量手绘，可导出 |
| **表格笔记** | 内嵌轻量表格 | 类 Notion 数据库 |

#### D.3 组织结构
```
工作区（Workspace）
└── 笔记本（Notebook）
    └── 分区（Section，可选）
        └── 笔记（Note）
            └── 版本历史（快照）
```
- 支持无限层级嵌套
- 支持跨笔记本引用 `[[笔记标题]]` 双向链接
- 全文搜索（含笔记内容）

#### D.4 标签与归档
- 多标签体系（颜色 + 名称）
- 归档操作：归档后不显示在主列表，但可搜索
- 回收站：删除后保留 30 天，可恢复

#### D.5 导入 / 导出
- 导入：.md / .txt / .docx / .html
- 导出：.md / .pdf / .html / .docx（单篇或批量导出）

---

### 模块 E：书库（Books）

**用途**：电子书上传、应用内阅读、文件管理、分类归档。

#### E.1 布局
```
[左：分类树 / 书架]       [右：书籍网格 / 列表]
  180px                    剩余宽度
```

#### E.2 支持格式
- **阅读类**：EPUB / PDF / MOBI / AZW3
- **文档类**：TXT / MD（用阅读器打开）
- **漫画类**：CBZ / CBR（后期支持）

#### E.3 书库管理
- 添加方式：拖拽文件或文件夹 / 点击导入
- 元数据：自动提取书名、作者、封面、语言（EPUB 优先）；PDF 提取标题
- 手动编辑元数据字段
- **文件管理器定位**：右键菜单 → "在文件管理器中显示"（调用系统 `reveal in Finder / Explorer`）

#### E.4 阅读器界面
- EPUB 阅读器：分页 / 滚动两种模式；字体大小、字体、行间距可调
- PDF 阅读器：基于 `PDF.js`；支持文本选择、缩放、跳页
- **阅读进度**：自动保存，下次打开继续
- **书签**：任意位置添加书签并备注
- **高亮与批注**：选中文本 → 高亮颜色 → 可加批注；批注汇总面板
- **目录面板**（TOC）：章节快速跳转

#### E.5 分类与归档
- **书架**：自定义书架（如"科技"、"文学"、"待读"）
- **阅读状态**：想读 / 在读 / 已读
- **标签系统**：与笔记共用全局标签
- **归档**：与笔记模块相同逻辑

---

### 模块 F：工具箱（Toolbox）

**用途**：各类小工具的统一入口，采用插件化设计。

#### F.1 布局
```
[左：工具分类列表]   [右：工具主界面]
  160px               剩余宽度
```
每个工具以独立"面板"形式打开，支持弹出为独立悬浮窗口。

#### F.2 内置工具清单

**⏱ 番茄钟 / 专注计时**
- 标准番茄：25分钟专注 + 5分钟休息
- 自定义时长（专注 / 短休 / 长休）
- 循环计数显示（今日完成几个番茄）
- 到时系统通知 + 可选音效
- 最小化为顶部悬浮小窗（始终置顶）

**💱 汇率 / 单位换算**
- 汇率：本地缓存汇率表（上次联网更新时间显示），离线可用
- 单位换算：长度 / 重量 / 温度 / 面积 / 速度 / 存储大小
- 界面：左侧输入 + 右侧单位选择 + 换算结果实时显示

**🔐 密码管理器**
- 本地加密存储（AES-256）
- 主密码解锁（应用启动时可设置自动锁定时长）
- 记录字段：网站名 / URL / 账号 / 密码 / 备注
- 密码生成器（自定义长度、字符规则）
- 搜索 / 分组 / 复制到剪贴板（复制后 30 秒自动清除剪贴板）
- ⚠️ 安全提示：数据完全本地，不联网，不同步（云同步版本需用户显式开启）

**🗂 后续可扩展工具（预留槽位）**
- 剪贴板历史
- Markdown 快速草稿
- JSON / Base64 / URL 编码解码
- 颜色拾取器
- 正则表达式测试器
- 网速测试
- ...（用户社区提交）

## 五、全局功能

### 5.1 全局搜索（`Cmd/Ctrl + K`）
- 搜索范围：任务标题 / 笔记全文 / 书名作者 / 视频标题 / 工具名称
- 结果分组显示（按模块）
- 支持快捷指令：输入 `/task 开会` 快速创建任务

### 5.2 标签系统
- 全局统一标签库，跨模块使用
- 标签颜色自定义
- 标签管理页（合并、重命名、删除）

### 5.3 数据存储结构
```
~/LifeOS/
├── database/
│   ├── tasks.db          (SQLite)
│   ├── notes.db          (SQLite)
│   ├── books.db          (SQLite，元数据)
│   ├── videos.db         (SQLite)
│   └── vault.db          (SQLite，AES加密，密码管理器)
├── files/
│   ├── notes/            (笔记附件)
│   ├── books/            (电子书原文件)
│   └── videos/           (下载的视频)
├── config/
│   └── settings.json     (用户设置)
└── backup/               (自动备份)
```
- **数据目录可自定义迁移**（设置中一键迁移）
- **自动备份**：可配置每日自动将 database/ 备份为压缩包，保留最近 N 份

### 5.4 设置模块（Settings）
| 分类 | 选项 |
|------|------|
| 外观 | 主题切换 / 字体大小 / 紧凑/宽松布局 |
| 语言 | 中文 / English |
| 通知 | 任务提醒开关 / 番茄钟提醒 / 通知方式 |
| 视频 | 下载路径 / 默认画质 / yt-dlp 更新 |
| 书库 | 默认书库路径 / 阅读器默认字体 |
| 数据 | 数据目录迁移 / 备份设置 / 导入导出 |
| 安全 | 密码管理器主密码 / 自动锁定时长 |
| 关于 | 版本号 / 检查更新 / 开源协议 |

### 5.5 系统托盘（System Tray）
- 最小化到托盘（不退出）
- 托盘菜单：显示今日任务数 / 快速新建任务 / 番茄钟状态 / 打开主窗口 / 退出

## 六、技术架构建议

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 桌面框架 | Electron + Vite | 主流稳定，生态完善 |
| 前端框架 | React + TypeScript | 组件化，类型安全 |
| 状态管理 | Zustand | 轻量，适合中型应用 |
| 本地数据库 | SQLite（better-sqlite3） | 快速、零配置 |
| 笔记编辑器 | Tiptap（富文本）+ CodeMirror（Markdown） | 灵活可扩展 |
| PDF 阅读 | PDF.js | 官方开源，稳定 |
| EPUB 阅读 | Foliate-js 或自研 | 开源 EPUB 引擎 |
| 视频下载 | yt-dlp（bundled binary） | 跨平台，更新活跃 |
| 视频播放 | mpv（IPC 控制）或 HTML5 Video | 格式兼容性强 |
| 数据加密 | Node.js crypto（AES-256-GCM） | 密码管理器加密 |
| 构建打包 | electron-builder | 三平台打包 |
| 自动更新 | electron-updater | 后期支持 |

## 七、界面原型设计要点（提示词参考）

以下内容可直接提供给 ChatGPT Image / Figma AI 进行原型图生成：

### 设计关键词
- 布局：`Left sidebar navigation, 60px collapsed / 200px expanded, three-column layout for notes module`
- 色彩（Minimal 主题）：`White background #FFFFFF, sidebar #F7F7F7, accent blue #4F46E5, text #1A1A1A`
- 色彩（Dark Tech 主题）：`Background #0D0D0D, neon accent #00F5D4, border glow effect, monospace font`
- 组件：`Rounded cards with subtle shadow, tag pills with color dots, progress bars, avatar initials`
- 特效：`Smooth sidebar expand transition, hover state card lift, active module highlight`

### 各模块原型优先级
1. **仪表盘**（Dashboard）—— 展示整体风格，最高优先
2. **笔记模块**（三栏布局）—— 核心功能，复杂度高
3. **任务看板视图**（Kanban）—— 视觉吸引力强
4. **书库网格 + 阅读器**—— 封面视觉
5. **工具箱主页**—— 卡片网格布局
