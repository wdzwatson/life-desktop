# D-01 macOS RC P0 用户旅程用例

日期：2026-07-17
任务：D-01
状态：已完成用例编写，待使用签名/安装包执行
目标版本：LifeOS v1.0 RC
优先平台：macOS

## 1. 执行规则

- 使用真实打包应用，不以 Vite 开发服务器代替安装包验收。
- 每条用例记录版本、操作系统、数据目录、结果、日志和截图路径。
- P0 用例任何数据丢失、阻塞性崩溃、无法启动或无法恢复都阻止 RC 进入内部试用。
- 数据操作前先创建可清理的测试用户和测试备份，不使用真实个人数据。

## 2. 账户与启动

| ID | 场景 | 操作 | 通过标准 |
|---|---|---|---|
| P0-01 | 首次启动 | 安装并启动应用，创建本地账户 | 窗口正常出现，账户创建成功，用户目录和数据库自动生成 |
| P0-02 | 密码登录 | 为账户设置密码，退出，重新启动并输入正确/错误密码 | 正确密码进入工作区；错误密码不泄露数据、不绕过验证 |
| P0-03 | 账户切换 | 创建第二个账户，退出并切换账户 | 两个账户数据库和文件隔离，当前用户数据不串读 |
| P0-04 | 重启保留 | 创建任务、笔记和设置后退出重启 | 主题、语言、账户和内容保持一致 |
